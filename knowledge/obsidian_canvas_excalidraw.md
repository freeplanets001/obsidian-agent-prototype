# Obsidian ビジュアル思考マスターガイド (Canvas と Excalidraw)

Obsidianはテキストベースのツールであるという印象が強いですが、ビジュアルに考えをまとめるための強力な機能やプラグインも備えています。テキストとビジュアルをシームレスに行き来することが、真の「第2の脳」の条件です。

---

## 1. 公式機能: Obsidian Canvas (キャンバス)

Obsidian v1.1.0 で導入された公式の無限ホワイトボード機能（マインドマップ/フローチャート機能）です。ファイル拡張子は `.canvas` となります。

### 1.1 Canvasの主な用途
- ブレインストーミング
- アイデアの視覚的な整理（インフォグラフィック化）
- 複数ノートの同時プレビュー・俯瞰（ダッシュボード的利用）
- プロジェクトの体系的なマップ作成（MOCの視覚化版）

### 1.2 Canvasの操作と機能
- **カードの追加:** ダブルクリックまたは右クリックからテキストカードを追加。
- **ノートの追加:** 左サイドバーまたはファイラーからMarkdownノートをドラッグ＆ドロップで配置。ノートはCanvas内で直接編集可能（双方向リンク対応）。
- **メディアの追加:** 画像、PDF、ウェブページ、YouTube動画などをドラッグ＆ドロップで配置可能。
- **コネクタ（矢印）:** カードやノートの端からドラッグして、別の要素へ矢印を引ける。矢印のラベルを付けたり、色を変更したりできる。
- **グループ化:** 複数要素を選択して `G` キーでグループ化し、背景色をつけて領域を分ける。
- **Markdown エクスポート:** Canvas全体を1枚のMarkdown画像として出力することはできないため、他のノートに埋め込む用途が基本。

### 1.3 Canvasの埋め込み (Embed)
他のMarkdownノートの中にCanvasを埋め込むことができます。

```markdown
![[プロジェクト計画.canvas]]
```
これにより、テキストドキュメントの中に視覚的な全体像を配置できます。

## 2. 最強の描画プラグイン: Excalidraw

Excalidraw は、手書き風のホワイトボードツールです。Obsidian版は信じられないほど多機能で、Canvas以上の機能（本格的な描画、スクリプト実行など）を持ちます。ファイル拡張子は標準の `.md` で保存されるため、Obsidian外でもデータが読み取れるという驚異的な仕様です。

### 2.1 主な特徴
- **手書き描画:** ペンツールによる自由な描画。Apple Pencil等と相性抜群。
- **図形ツール:** 四角、丸、ひし形、線、矢印などを素早く配置。すべてが手書き風（ラフ）な線になります。
- **テキスト:** Obsidian内のノートへのリンク `[[ノート名]]` を入力すると、クリックでノートへ飛べる。
- **Markdownとの統合:** 描画データが見えない形でMarkdownファイルのテキスト末尾に保存されるため、ノートの一部として描画データを扱える。

### 2.2 Excalidrawの活用法

#### 2.2.1 スケッチノート (Sketchnoting)
会議中や動画視聴中に、文字だけで追いつかない関係性や空間的な配置を含んだメモを「描き」ながら「書く」ことができます。

#### 2.2.2 画像へのアノテーション（書き込み）
画像ファイルをExcalidrawにドラッグし、その上に赤ペンで丸をつけたり、矢印を引いて解説を入れたりできます。これは標準のObsidianにはない強力な機能です。

#### 2.2.3 PDFへの書き込み
PDFファイルを挿入し、特定のページの上にメモを描画できます。論文や仕様書のレビューに最適です。

### 2.3 Excalidraw Scripts
コミュニティが作成した自動化スクリプトを実行できます。
- 要素を整列させるスクリプト
- 特定のMarkdownファイルの構造を自動でマインドマップ化するスクリプト

## 3. Canvas vs Excalidraw: どちらを使うべきか？

どちらもホワイトボードツールですが、用途が明確に分かれます。

| 特徴 | Obsidian Canvas | Excalidraw プラグイン |
| :--- | :--- | :--- |
| **強み** | **ノートの俯瞰**、YouTubeやウェブの貼り付け、公式機能ゆえの安定性 | **手書き、図解**、PDFアノテーション、Apple Pencilでの思考 |
| **弱み** | 自由な線（ペン）で絵は描けない。 | キャンバス上に大量のMarkdownノートを並べて中身を読むのには不向き。 |
| **ファイル形式** | `.canvas` (独自JSON形式) | `.md` (Markdownファイル内にデータ埋め込み) |
| **おすすめ用途** | リサーチボード、ムードボード、大量の資料を並べての執筆 | 手書きメモ、概念図の作成、PDFへの直接の赤入れ |

**結論:** 「すでに存在するノートや資料を並べて整理する」なら **Canvas**。「図や絵をゼロから描いて考えをまとめる」なら **Excalidraw** を使いましょう。両方を使い分けるのが最も効率的です。
